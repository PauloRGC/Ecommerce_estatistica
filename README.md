# Atividade do curso de ADD EBAC

## Downloads necessários
- pandas
- seaborn
- matplotlib

## Descrição da atividade
A atividade consiste em pegar dados pré-estabelecidos e criar gráficos em cima deles, utilizando as bibliotecas citadas acima. O objetivo é praticar leitura de dados, manipulação com pandas e criação de visualizações com seaborn e matplotlib.
